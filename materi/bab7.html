7.1 Best Practice Struktur File & Folder

Struktur proyek yang rapi adalah fondasi untuk pemeliharaan kode dan kolaborasi tim.

A. Separation of Concerns

Kode HTML, CSS, dan JavaScript wajib dipisahkan ke dalam file-file terpisah (.html, .css, .js). Ini meningkatkan keterbacaan, caching, dan kemudahan debugging.

B. Penempatan JS

Letakkan tag <script> tepat sebelum tag penutup </body> atau gunakan atribut defer agar HTML termuat sempurna sebelum dimanipulasi.

    e.preventDefault(); // Stop Reload

    let isValid = true;

    // Cek Panjang Password
    if (pass.value.length < 8) {
        showError(pass, "Min 8 char!");
        isValid = false;
    }

    // Cek Kecocokan
    if (pass.value !== confirm.value) {
        showError(confirm, "Tidak cocok!");
        isValid = false;
    }

    if (isValid) showSuccess();
});
